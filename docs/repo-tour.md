# Repository Tour

## Root files
- `README.md`: polished landing page for GitHub visitors
- `CITATION.cff`: citation metadata
- `LICENSE`: open-source license text
- `requirements.txt`: lightweight Python dependency list
- `.gitignore`: common Python and neuroimaging ignores

## Folders
- `docs/`: polished summaries and profile-ready text
- `reports/`: original course submission files
- `.github/`: templates for collaboration and maintenance

## Best use case
Use this repo as a public portfolio project, a graduate application artifact, or a supporting repository linked from your resume.
